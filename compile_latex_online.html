Free Online LaTeX Compilers (No Quota Limits)

Option 1: LaTeX Base (Recommended - No Sign Up)

1. Go to: latexbase.com
2. Delete their sample code
3. Copy your main_extended.tex content
4. Click "Compile" (no account needed!)
5. Download PDF directly

Option 2: Papeeria (Free Plan)

1. Visit: papeeria.com
2. Create free account (just email)
3. Upload your paper folder
4. Compile and download

Option 3: ShareLaTeX Clone

1. Try: writelatex.com
2. Free tier available
3. Upload and compile

Option 4: CoCalc (Limited Free)

1. Go to: cocalc.com
2. Create free account
3. New LaTeX document
4. Paste your code

Quick Local Option: TinyTeX

1. Download: TinyTeX installer
2. Run: install-windows.bat
3. Then in your paper folder run:
tinytex main_extended.tex
4. Creates PDF instantly!
